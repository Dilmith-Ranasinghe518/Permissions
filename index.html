Permissions & Data Usage

Last Updated: 07 May 2025

Notification Access

Parent Pal AI requires notification access to help parents receive important child-related safety alerts and reminders.

  • View safety notifications from the child’s device
  • Receive important alerts in real time

Parent Pal AI does not read personal messages, store notification content, or share notification data with third parties.

Display Over Other Apps

This permission allows Parent Pal AI to show important child safety alerts on top of other apps.

  • Ensure emergency alerts are immediately visible
  • Prevent missing scheduled child reminders

This permission is used only for safety and parental alerts, not for ads or background pop-ups.

Device Administrator Access

Device administrator access is required to enable essential parental control and child safety features.

  • Enforce safety settings
  • Prevent unauthorized app removal
  • Manage child device restrictions

No device changes are performed without explicit parent authorization.

App Usage Access

App usage access helps parents understand how the child uses apps and manage screen time responsibly.

  • View app usage summaries
  • Monitor screen-time patterns
  • Support healthy digital habits

Parent Pal AI does not access personal app content or monitor activity beyond parental supervision features.

Location Access

Location access is used to help parents view the child’s device location for safety purposes.

  • Enable real-time child location viewing
  • Support safety alerts and location-based features

Location data is accessible only to the linked parent account and is not shared with third parties. Location may be collected even when the app is not actively in use to support safety features.

Exact Alarm / Event Alerts

Exact alarm permission is used to deliver scheduled child-related reminders and safety alerts at the correct time.

This permission is used only for reliable event alerts and is not used for advertising or unrelated purposes.
